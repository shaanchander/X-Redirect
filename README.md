# X Redirect

X Redirect is a Firefox extension that replaces Twitter/X links with xcancel links.

## Firefox Add-ons

Install from Mozilla Add-ons:
https://addons.mozilla.org/en-US/firefox/addon/x-redirect/

## What It Does

- Detects links to x.com and twitter.com on web pages
- Rewrites those links to point to xcancel.com
- Runs continuously so links added dynamically are also updated
